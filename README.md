# Diabetes-Prediction-using-ML
This project aims to predict whether a person is likely to have diabetes based on various medical attributes such as glucose level, blood pressure, insulin level, BMI, and more. Using Machine Learning algorithms, the model analyzes health data to provide accurate and early predictions that can help in medical decision-making and awareness.
